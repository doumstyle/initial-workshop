Iteration 1 - Inline vs Block elements

Write a paragraph that contains some span or em tags of different colors. Spans are inline elements, meaning that their default display property value is inline. Those elements will follow the horizontal flow of the document. Inline elements go next to another inline element so you won't have any line break until you decide to insert one (like here withe the br tag).
Images are inline elements too ! That's a simple but super important point to remember ! Feel free to dig deeper with the MDN doc.
